# 两段式接口

## 概述

基于单算子API执行方式调用算子API时，通常分为"两段式"，是 aclnn 算子调用的核心模式。

## 接口模式

```cpp
// 第一段：计算 workspace 大小
aclnnStatus aclxxXxxGetWorkspaceSize(
    const aclTensor *src, ...,
    aclTensor *out, ...,
    uint64_t *workspaceSize,
    aclOpExecutor **executor);

// 第二段：执行计算
aclnnStatus aclxxXxx(
    void *workspace,
    uint64_t workspaceSize,
    aclOpExecutor *executor,
    aclrtStream stream);
```

## 调用流程

1. 调用第一段接口 `aclxxXxxGetWorkspaceSize`，计算所需 workspace 内存大小
2. 按 `workspaceSize` 申请 NPU 内存
3. 调用第二段接口 `aclxxXxx` 执行计算

## 命名约定

- `aclxx` = 算子接口前缀（如 `aclnn`）
- `Xxx` = 算子类型（如 `Add`）

## 关键规则

- **workspace**：除输入/输出外，API 在 AI 处理器上完成计算所需要的临时内存
- **第二段接口不可重复调用**，如下调用方式会异常：

```cpp
aclxxXxxGetWorkspaceSize(...)
aclxxXxx(...)
aclxxXxx(...)  // 异常
```

- `aclOpExecutor` 在一阶段接口中自动创建，二阶段接口调用后自动释放
