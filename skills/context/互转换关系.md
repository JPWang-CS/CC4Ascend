# 互转换关系（Type Conversion）

## 概述

当 API 输出的 `aclTensor` 数据类型与输入推导后的计算类型不一致时，API 内部将计算结果转换为输出类型。

## 转换规则

### 支持转换

| 类型族 | 族内转换 | 往浮点转 | 往复数转 |
|--------|:---:|:---:|:---:|
| 浮点 (FLOAT16, FLOAT, DOUBLE, BF16) | ✓ | — | ✓ |
| 整数 (INT8/16/32/64, UINT8/16/32/64) | ✓ | ✓ | ✓ |
| 复数 (COMPLEX64, COMPLEX128) | ✓ | — | — |
| BOOL | — | ✓ | ✓ |

### 不支持

除上述场景外，其他类型间不可互转，调用 API 时会参数校验失败。

## 关键记忆

- **整数 → 浮点**：允许（如 INT32 → FLOAT16）
- **浮点 → 整数**：不允许（会丢失精度）
- **整数 ↔ 整数**：任意互转
- **浮点 ↔ 浮点**：任意互转
- **BOOL → 整数/浮点/复数**：允许
